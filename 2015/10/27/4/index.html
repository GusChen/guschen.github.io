发表于 2015-10-27   |  

基础语法

  • 区分大小写，在js中test和Test是完全不一样的。

  • 标识符，就是指变量、函数、属性的名字，或者函数的参数，第一个字符必须是一个字母一个字母、下划线_或一个美元符号$，其他字符可以是字母、下划线、美元符号或数字。ECMAScript 标识符采用驼峰大小写格式例如：firstSecond

  • 注释

  • 严格模式是为 JavaScript 定义了一种不同的解析与执行模型

  • 语句，在编写代码中不推荐省略;和{}
if (test)
  alert(test); // 有效但容易出错，不要使用
if (test){ // 推荐使用
  alert(test);
}
