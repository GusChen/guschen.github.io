CSS3 animation属性

animation包含以下属性
属性名 说明
animation-name 动画的名称
animation-duration 规定完成动画所花费的时间，以秒或毫秒计,例如.1s、1s。
animation-timing-function 规定动画的速度曲线,默认为ease,可选参数linear、 ease、 ease-in、ease-out、ease-in-out、cubic-bezier(n,n,n,n)、steps([n┃start ┃ end],[n┃start┃end])。
animation-delay 规定动画何时开始,默认为0s
animation-iteration-count 规定动画被播放的次数,默认为1，infinite则为循环播放。
animation-direction 规定动画是否在下一周期逆向地播放。默认是 normal，alternate则逆向播放。
animation-play-state 规定动画是否正在运行或暂停。默认是 “running”，paused则暂停，可以通过js控制。
animation-fill-mode 规定对象动画时间之外的状态。默认为none,可选参数有none 、 forwards 、 backwards 、 both

参考w3school

keyframes的语法规则

@keyframes animationname {keyframes-selector {css-styles;}}

animationname 必需。定义动画的名称。
keyframes-selector 0-100%、from（与0% 相同）to（与 100% 相同）
css-styles 一个或多个合法的 CSS 样式属性。
